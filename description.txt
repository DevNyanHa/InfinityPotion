Infinity Potion [BETA]

This mod makes all consumable items-such as potions, boss summoning items, and ammo-unlimited.
If you want to change the mod settings, you can easily do so at [Settings->Mod Settings->InfinityPotion->Main Config].

Features
- You can use it without any crafting hassle-as long as you have just one consumable item
- All consumable items, including potions, summoning items, and ammo, can be used infinitely.
- You can customize which items are affected in the settings.
- Items with the [Infinity] tag in their description can be easily identified.

This mod is server-side.
